# StatusIPS

Status IPS

## Installation

Follow the steps to run the project

Step 1: 
Create the virtual environment
```bash
python3 -m venv .venv
```
Step 2:
Activate the VM
```bash
source ./.venv/bin/activate
```
Step 3
Install all dependencies
```bash
pip install -r requirements.txt
```

Step 4
Run the script
```bash
python main.py
```
